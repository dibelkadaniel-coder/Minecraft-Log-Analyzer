# Minecraft-Log-Analyzer

Easy to run and install.


## Create Gemini API Key
- Create account: [https://aistudio.google.com](https://aistudio.google.com)
- Click to button Get Api Key
- Copy API Key and enter it on `GEMINI_API_KEY`
  
## Install Dependency
- Run `pip install google-genai fastapi uvicorn pydantic`
- Run `uvicorn main:app --reload`

## Final
- Enter the html website (On local [http://127.0.0.1:5500](http://127.0.0.1:5500/index.html))
